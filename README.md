# tool-dashoard
Un sito che permette di usare dei tool per semplificare il lavoro. I tool presenti sono:
* QR code generator;
* webP converter;
* IP info;
* Gestore squadra;
## QR code generator
Questo tool permette di creare dei codici QR usando un link per aprire una pagine web. Il codice generato può anche essere scaricato sottoforma di immagine **JPG** o **PNG**. È importante che il link inizi con **http**, altrimenti ci sarà un errore.
## webP converter
Questo tool permette di convertire delle immagini **.webp** in un formato di tipo **.jpg** o **.png**
## IP info
Qui si possono ottenere alcune informazioni sul proprio indirizzo IP pubblico, quindi indirizzo ipv4 pubblico e posizione geografica (non troppo affidabile). È in grado di restiruire regione, città e latitudine e longitudine.
## Gestore squadra
Un tool che permette di organizzare la formazione di due squadre di calcio partendo da 8 giocatori fino a 14. Si inseriscono prima tutti i nomi e poi si procede allo smistamento.
